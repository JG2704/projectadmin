Backend (FastAPI + SQLite servidor)

1) Entrar a la carpeta backend:
   cd hotelmascotas/backend

2) Levantar API:
   uvicorn main:app --host 0.0.0.0 --port 8000 --reload

3) Inicialización de base:
   - La API valida en startup `database/petlodge_backend.db`.
   - Si la base no existe o no tiene tablas, carga automáticamente:
     - database/schema.sql
     - database/seed.sql

4) Credenciales de prueba mínimas:
   - correo: cliente@hotel.com
   - contraseña: 123456

Notas:
- La base oficial compartida vive en backend/database/petlodge_backend.db.
- La SQLite local del frontend se mantiene solo como referencia del modelo.
